# Veri

Bu klasör veri dosyalarını içermelidir ancak Kaggle kuralları gereği repoya eklenmemiştir.

## İndirme

Kaggle yarışma sayfasından aşağıdaki dosyaları indirip bu klasöre koyun:

```
data/train.csv
data/test_x.csv
data/sample_submission.csv
```

## Açıklama

| Dosya | Açıklama |
|---|---|
| `train.csv` | Eğitim verisi — hedef değişken dahil |
| `test_x.csv` | Test verisi — tahmin yapılacak satırlar |
| `sample_submission.csv` | Submission format örneği |

**Hedef değişken:** `bilissel_performans_skoru`  
**Metrik:** RMSE
